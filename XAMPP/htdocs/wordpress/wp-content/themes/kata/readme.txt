== Description ==
Kata is a modern full-featured WordPress theme developed with cutting-edge web technology that makes it easier than ever to create beautiful, fast-loading and accessible websites.

Styler is the most important tool in Kata as it allows you to customize your website using a simple and straightforward user interface.
You can change fonts, backgrounds, colors, and almost any other existing element in the website using the Styler.

Kata theme makes it much faster to create any website since it includes a lot of ready-made demos.

Kata theme makes customizing WooCommerce pages easier than ever with a seamless integration built into it. It has unique WooCommerce features and beautifully designed templates to build your online store easily.



== Frequently Asked Questions ==

**Does this theme support any Elementor?**

Kata is completely compatible with Elementor.
